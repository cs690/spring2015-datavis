    Visualization Of YouTube Dataset

    This project is about visualizing YouTube dataset. We will implement a set of linked and interactive visualizations referring to a paper, GPLOM. Our core functionality is to provide users with a web interface and allow them to be able to explore the data through it. For example, users may find what are the top 3 categories viewed, or what is the most popular video rated in a certain day. The interface will be composed of a set of functionality that the paper describes such as filtering, linking, or text search.

    Figure 1. Design Overview

    In this project, there are a few components: a website to put our visualizations, a preprocessed dataset, and a JavaScript library. As figure 1 shows, we will first load and then visualize the dataset using D3, and hence put it on our website. We are trying to use GPLOM (Generalized Plot Matrix) for our visualizations. It is a variant of scatterplot matrices, and also a matrix to show multiple features. That is to say, we need to implement the following modules first: Scatterplot, barchart and heatmap. Each of them should be resizable and reusable. Another module, controller, manages them and constructs the matrix, setting up proper chart types to different combinations of feature values.
