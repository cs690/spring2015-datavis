    D3.js

    D3.js is a JavaScript library for manipulating documents based on data. We are using the latest v3.4.12 version of it.

    D3.js is a JavaScript library especially suited to visualizing data in a dynamic and interactive manner. We chose the library since it allows us to create interactive visualizations easily without the use of additional software or libraries; this aspect enables the user to see the visualization without installing any software and also helps us to deploy the visualization easily in our website.

    CS690 Master Project, Spring 2015. Computer Science Department of University of San Francisco.
